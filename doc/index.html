Examples

Bloombees theme includes three main type of buttons, each comming in three variants: normal and disabled

<button class="button button-main" type="submit">Main CTA</button>
<button class="button button-main" type="submit" disabled>Main CTA</button>

<button class="button button-default" type="submit">Action in page</button>
<button class="button button-default" type="submit" disabled>Action in page</button>

<button class="button button-second" type="submit">Second CTA</button>
<button class="button button-second" type="submit" disabled>Second CTA</button>

Buttons with icons

<button class="button button-main" type="submit">Main CTA <i class="bb-icon-bee"></i></button>
<button class="button button-default" type="submit">Second CTA <i class="bb-icon-facebook-in-o"></i></button>
<button class="button button-second" type="submit">Logo Bee <i class="bb-icon-check"></i></button>
